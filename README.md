# Planetarf Landing

Next.js 15 + React 19 project.

## Package Manager

This repository uses **Yarn**.  
Use `yarn.lock` as the single source of truth for dependencies.

## Getting Started

Install dependencies:

```bash
yarn install
```

Run development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

## Available Scripts

```bash
yarn dev
yarn build
yarn start
yarn lint
```
