# Project: Planetarf

## Rules

- NEVER create git worktrees or new branches unless explicitly asked.
- NEVER use GSD worktree mode. Work directly on the current branch.
- Do NOT create branches automatically at session start.

## Dev Server

The project uses Next.js. To start the dev server use preview_start with "Next.js Dev" from `.claude/launch.json`. Node is at `/usr/local/bin/node`.

If port 3000 is busy, set `autoPort: true` — do not ask the user.
